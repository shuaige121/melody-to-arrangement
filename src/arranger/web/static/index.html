Melody to Arrangement

The FastAPI server is running, but the built React frontend is not present in this environment.

Run cd web && npm install && npm run build and restart arranger serve to serve the full DAW interface.

Available Now

  • POST /api/upload/midi parses uploaded MIDI and returns melody notes, tempo, and time signature.
  • POST /api/arrange accepts a MIDI upload and returns an arranged MIDI file.
  • POST /api/upload/audio is intentionally unavailable in this build and returns 501.

Notes

  • MIDI and MusicXML are the reliable input paths today.
  • The browser app includes lightweight monophonic audio import, but there is no server-side audio transcription API in this repository.
