# 03-节奏与鼓组模式

## 0. 二进制定义基线
- 固定通道: `channel=9` (0-indexed, 即 MIDI Channel 10)
- 事件: `note_on(0x90)` / `note_off(0x80)`
- 主参数: `note_number(0-127)`, `velocity(0-127)`, `tick(delta_time)`
- 本文 16 步网格: 1 小节 4/4, `PPQ=480`, 每步 `120 tick`。
- 本文 12 步 Swing 网格: 1 小节 4/4 三连网格, 每步 `160 tick`。

## 2. Velocity 力度层级 (二进制量化)
- `Ghost Note`: `velocity=30-50`
- `Normal`: `velocity=80-100`
- `Accent`: `velocity=110-127`

力度层级对应效果:
- `30-50`: 低能量、弱重音、衔接/粘合律动
- `80-100`: 主体击打、稳定 groove 信息密度
- `110-127`: 强重音、结构锚点（反拍、段落边界、Crash 落点）

## 5. 时间量化 (PPQ=480)

### 5.1 音符时值 -> tick
- 全音符: `1920`
- 二分音符: `960`
- 四分音符: `480`
- 八分音符: `240`
- 十六分音符: `120`
- 三十二分音符: `60`
- 四分三连: `320`
- 八分三连: `160`

### 5.2 BPM 与 MIDI Tempo 二进制
- Set Tempo Meta Event: `FF 51 03 tt tt tt`
- `tt tt tt` = `microseconds_per_quarter_note (MPQN)`
- 公式: `MPQN = 60000000 / BPM`
- 公式: `BPM = 60000000 / MPQN`
- 示例
  - `60 BPM -> 1000000 us/qn`
  - `80 BPM -> 750000 us/qn`
  - `90 BPM -> 666667 us/qn`
  - `100 BPM -> 600000 us/qn`
  - `120 BPM -> 500000 us/qn`
  - `128 BPM -> 468750 us/qn`
  - `140 BPM -> 428571 us/qn`

### 5.3 Swing tick 偏移
以八分对 (`1/8 + 1/8`) 为例，基准总长 `480 tick`:
- 直八分 (`1:1`): `240 + 240`，第二个八分起点 `+240`
- Swing `2:1`: `320 + 160`，第二个八分起点 `+320`，相对直拍偏移 `+80`
- Swing `3:2`: `288 + 192`，第二个八分起点 `+288`，相对直拍偏移 `+48`
- 重 Swing `5:3`: `300 + 180`，第二个八分起点 `+300`，相对直拍偏移 `+60`
