# 第一部分：MIDI 二进制指标完整定义

## 1. 理论名称（中英）→ MIDI 二进制参数 → 数值范围/公式

- `MIDI 通道语音消息 (Channel Voice Message)` → `status_byte(0x8n-0xEn) + data_byte` → `n=0..15` 表示通道。
- `标准 MIDI 文件事件 (SMF Event)` → `delta_time(VLQ) + event_bytes` → `delta_time` 为可变长数量（1-4 字节）。
- `运行状态 (Running Status)` → 省略重复 `status_byte`，沿用上一个通道状态字节 → 只影响编码长度，不改变语义。

## 2. Status Byte + Data Byte 完整映射（编曲相关主表）

| 理论名称（中英） | Status Byte | Data Byte 1 | Data Byte 2 | 二进制定义 |
|---|---:|---:|---:|---|
| 音符关 / Note Off | `0x8n` | `note_number(0-127)` | `velocity(0-127)` | 结束音符 |
| 音符开 / Note On | `0x9n` | `note_number(0-127)` | `velocity(0-127)` | 启动音符；`velocity=0` 等价 Note Off |
| 复音触后 / Polyphonic Key Pressure | `0xAn` | `note_number(0-127)` | `pressure(0-127)` | 按键后压（逐音） |
| 控制器 / Control Change | `0xBn` | `controller(0-127)` | `value(0-127)` | 连续控制（CC） |
| 音色切换 / Program Change | `0xCn` | `program(0-127)` | - | GM 音色号切换 |
| 通道触后 / Channel Pressure | `0xDn` | `pressure(0-127)` | - | 整通道后压 |
| 弯音 / Pitch Bend Change | `0xEn` | `LSB(0-127)` | `MSB(0-127)` | 14-bit 值：`bend = LSB + 128*MSB`，范围 `0..16383`，中心 `8192` |

## 3. Channel Mode（通过 CC 120-127 编码）

| 理论名称（中英） | Status Byte | Data1 | Data2 |
|---|---:|---:|---:|
| 全音符关 / All Sound Off | `0xBn` | `120` | `0` |
| 重置全部控制器 / Reset All Controllers | `0xBn` | `121` | `0` |
| 本地控制 / Local Control | `0xBn` | `122` | `0/127` |
| 全音符释放 / All Notes Off | `0xBn` | `123` | `0` |
| Omni Off/On, Mono/Poly | `0xBn` | `124..127` | `0` 或声部数 |

## 4. System Message（0xF*）

| 理论名称（中英） | Status Byte | 数据结构 |
|---|---:|---|
| 系统独占开始 / SysEx Start | `0xF0` | `manufacturer + payload + 0xF7`（在线流）；SMF 中写为 `F0 + length + data` |
| MIDI Time Code Quarter Frame | `0xF1` | 1 字节 |
| Song Position Pointer | `0xF2` | 2 字节（14-bit） |
| Song Select | `0xF3` | 1 字节 |
| Tune Request | `0xF6` | 无数据 |
| SysEx End / Escape | `0xF7` | 结束或转义 |
| Timing Clock | `0xF8` | 无数据 |
| Start / Continue / Stop | `0xFA/0xFB/0xFC` | 无数据 |
| Active Sensing | `0xFE` | 无数据 |
| System Reset | `0xFF` | 在线流为系统复位；SMF 中 `0xFF` 作为 Meta 前缀 |

## 5. SMF Meta Event（`FF type length data`）

| 理论名称（中英） | 字节结构 | 关键参数 |
|---|---|---|
| 轨道结束 / End of Track | `FF 2F 00` | 轨道终止 |
| 速度 / Set Tempo | `FF 51 03 tt tt tt` | `mpqn`（每四分音符微秒数，24-bit 大端） |
| 拍号 / Time Signature | `FF 58 04 nn dd cc bb` | `numerator=nn`；`denominator=2^dd` |
| 调号 / Key Signature | `FF 59 02 sf mi` | `sf=-7..+7`（降/升号数）；`mi=0大调/1小调` |
| SMPTE Offset | `FF 54 05 hr mn se fr ff` | SMPTE 时间起点 |
| 文本类元事件 | `FF 01..09 len text` | 标注文本（非音高控制） |

## 6. 栅栏定义（RAG Fence）

后续所有乐理都必须降维为以下参数之一：
- `note_on/note_off` (`0x90/0x80`)
- `note_number` (`0..127`)
- `velocity` (`0..127`)
- `channel` (`0..15`)
- `delta_time/tick`（VLQ 编码 tick）
- `control_change` (`0xB0`, `cc# + value`)
- `program_change` (`0xC0`, `program#`)
- `pitch_bend` (`0xE0`, 14-bit)

不可映射到上述二进制参数的概念，视为噪音并剔除。

# 第二部分：音阶与音高 → note_number 映射

## 1. 十二平均律（Twelve-Tone Equal Temperament）

- `音级类 / Pitch Class (PC)` → `pc = note_number mod 12` → 范围 `0..11`。
- `八度映射 / Octave Mapping` → `note_number = 12*(octave+1) + pc`（以 `C-1=0` 为基准）。
- `MIDI 全域`：`0..127`，对应约 `C-1..G9`。

| PC | 音名（等音） |
|---:|---|
| 0 | C |
| 1 | C# / Db |
| 2 | D |
| 3 | D# / Eb |
| 4 | E |
| 5 | F |
| 6 | F# / Gb |
| 7 | G |
| 8 | G# / Ab |
| 9 | A |
| 10 | A# / Bb |
| 11 | B |

锚点（常用）：`C4=60`，`A4=69`，`C5=72`。

## 2. 音阶（Scale）偏移数组定义

定义：给定主音 `root_pc`，音阶集合
`Scale(root) = { (root_pc + offset_i) mod 12 }`。

| 理论名称（中英） | MIDI 参数 | 偏移数组（半音） |
|---|---|---|
| 大调 / Major (Ionian) | `note_number mod 12` | `[0,2,4,5,7,9,11]` |
| 自然小调 / Natural Minor (Aeolian) | `note_number mod 12` | `[0,2,3,5,7,8,10]` |
| 和声小调 / Harmonic Minor | `note_number mod 12` | `[0,2,3,5,7,8,11]` |
| 旋律小调上行 / Melodic Minor (Asc) | `note_number mod 12` | `[0,2,3,5,7,9,11]` |

# 第四部分：拍号与时间 → delta_time/tick 映射

## 1. 时间轴核心参数

- `分辨率 / PPQ` → SMF Header `division`（当最高位为 0）→ `ticks_per_quarter = PPQ`。
- `速度 / Tempo` → `FF 51 03 b1 b2 b3` → `mpqn = (b1<<16) + (b2<<8) + b3`（微秒/四分音符）。
- `拍号 / Time Signature` → `FF 58 04 nn dd cc bb`：
  - `numerator = nn`
  - `denominator = 2^dd`
  - `cc`（每次节拍器点击的 MIDI Clock 数）
  - `bb`（每个四分音符包含的 32 分音符数，常用 `8`）

## 2. BPM 与时间换算

- `mpqn = 60,000,000 / BPM`
- `seconds_per_tick = mpqn / (PPQ * 1,000,000)`
- `seconds(delta_ticks) = delta_ticks * mpqn / (PPQ * 1,000,000)`

例：`BPM=120` 时，`mpqn=500000`，Set Tempo 字节为 `FF 51 03 07 A1 20`。

## 3. 每拍/每小节 tick 公式

- `ticks_per_beat = PPQ * (4 / denominator)`
- `ticks_per_bar = numerator * PPQ * (4 / denominator)`

以 `PPQ=480` 为例：

| 拍号（Time Signature） | 每拍 tick | 每小节 tick |
|---|---:|---:|
| 4/4 | 480 | 1920 |
| 3/4 | 480 | 1440 |
| 2/4 | 480 | 960 |
| 6/8 | 240 | 1440 |
| 12/8 | 240 | 2880 |

4/4 的二进制拍号事件常用：`FF 58 04 04 02 18 08`。

# 第五部分：调号 → note_number 集合（mod 12）

## 1. Key Signature 元事件到调性参数

- `调号 / Key Signature` → `FF 59 02 sf mi`
- `sf`：`-7..+7`（负数=降号，正数=升号）
- `mi`：`0=Major`，`1=Minor`

## 2. 五度圈调号映射为合法音级集合（Pitch Class Set）

定义：
- `合法音符集合 = { n | n mod 12 ∈ pitch_class_set }`
- 同一 `sf` 下，大调与关系小调共享同一 `pitch_class_set`，`mi` 只标识主音语义。

| sf | 大调 Major | 小调 Minor | pitch_class_set (mod12) |
|---:|---|---|---|
| -7 | Cb | Abm | `[1,3,4,6,8,10,11]` |
| -6 | Gb | Ebm | `[1,3,5,6,8,10,11]` |
| -5 | Db | Bbm | `[0,1,3,5,6,8,10]` |
| -4 | Ab | Fm | `[0,1,3,5,7,8,10]` |
| -3 | Eb | Cm | `[0,2,3,5,7,8,10]` |
| -2 | Bb | Gm | `[0,2,3,5,7,9,10]` |
| -1 | F | Dm | `[0,2,4,5,7,9,10]` |
| 0 | C | Am | `[0,2,4,5,7,9,11]` |
| +1 | G | Em | `[0,2,4,6,7,9,11]` |
| +2 | D | Bm | `[1,2,4,6,7,9,11]` |
| +3 | A | F#m | `[1,2,4,6,8,9,11]` |
| +4 | E | C#m | `[1,3,4,6,8,9,11]` |
| +5 | B | G#m | `[1,3,4,6,8,10,11]` |
| +6 | F# | D#m | `[1,3,5,6,8,10,11]` |
| +7 | C# | A#m | `[0,1,3,5,6,8,10]` |

## 3. 编曲约束公式（可直接用于生成）

- 音阶约束：`n mod 12 ∈ Scale(root_pc)`
- 调号约束：`n mod 12 ∈ KeySignature(sf)`
- 音程约束：`n2 - n1 = target_interval`
- 节奏约束：`on_tick/off_tick` 为整数 tick，且按 `PPQ + Tempo` 转换到秒
- 表情约束：力度用 `velocity`，长音踏板用 `CC64`，弯音用 `pitch_bend`

以上即“可二进制化”的实用乐理基础全集。
